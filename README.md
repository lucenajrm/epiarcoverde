# epiarcoverde

Repositório institucional de projetos de Vigilância em Saúde Digital do município de Arcoverde – Pernambuco.

Este repositório concentra iniciativas voltadas à modernização da Vigilância Epidemiológica, com foco em:
- uso qualificado de dados dos sistemas oficiais do SUS (SIM, SINAN, SINASC);
- integração com bases do IBGE;
- automação de rotinas;
- transparência, segurança da informação e apoio à tomada de decisão.
